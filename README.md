# 🚀 Backend Java Roadmap

Este repositório documenta minha jornada de evolução como desenvolvedor backend com foco em **Java**, com objetivo de atingir nível profissional comparável a grandes empresas de tecnologia.

O foco principal é construir uma base sólida em **engenharia de software**, com aplicação prática de **arquitetura**, **boas práticas** e **padrões utilizados em sistemas escaláveis**.

---

## 🎯 Objetivo

* Tornar-me desenvolvedor backend Java
* Construir projetos com qualidade de engenharia (nível FAANG)
* Dominar Spring Boot e arquitetura de sistemas
* Evoluir futuramente para fullstack
